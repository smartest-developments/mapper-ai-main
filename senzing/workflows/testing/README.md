# Automated Management Testing

## Purpose

Run 10 automated management-oriented test cases to compare Senzing outcomes against internal EPG grouping logic.

## Script

- `run_management_tests.py`

## Input

Provide one Senzing run directory generated by the E2E runner. Required files:

- `input_normalized.jsonl`
- `comparison/entity_records.csv`
- `comparison/matched_pairs.csv`

## Command

```bash
python3 senzing/workflows/testing/run_management_tests.py \
  /path/to/senzing_run_folder
```

Optional baseline comparison for TC-10:

```bash
python3 senzing/workflows/testing/run_management_tests.py \
  /path/to/senzing_run_folder \
  --baseline-metric Entity_EPG_Purity_Rate \
  --baseline-value 0.932
```

## Output

Created in `<run_dir>/testing/`:

- `management_test_results.json`
- `management_test_results.csv`
- `management_test_summary.md`
